**BOT PORTFOLIO**
Все команды, который сейчас доступны в боте:
/new_project - используй для добавления нового проекта
/skills - добавляет к проекту определнный скил
/projects - показывает все твои проекты
/delete - удаляет нужный проект
/update_projects - изменяет проект
